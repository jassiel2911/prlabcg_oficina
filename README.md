# Diorama de la vida cotidiana.

Consiste de un ambiente virtual interactivo representando una escena con elementos característicos de la vida cotidiana de varios
personajes de ficción.

- Geometría.
- Avatar.
- Recorrido.
- Iluminación.
- Animación.
